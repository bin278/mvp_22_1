## 使用说明

### 打印源代码文档

1. **转换为PDF**（推荐）
   ```bash
   # 使用文本编辑器打开文件
   # 选择"打印" -> "另存为PDF"
   # 或在命令行使用工具转换
   ```

2. **打印要求**
   - 纸张: A4
   - 字体: 等宽字体（Courier New、Consolas、Monaco等）
   - 字号: 10-12pt
   - 行距: 单倍行距
   - 页边距: 标准边距（上下2.5cm，左右2cm）
   - 页码: 连续编号（第1页到第60页）

3. **装订要求**
   - 左侧装订
   - 页码连续
   - 前后30页分别装订或合并装订均可

### 提交要求

根据中国版权保护中心的要求：
- 源代码文档需要清晰可读
- 页码需要连续编号
- 建议双面打印以节省纸张
- 每页都需要包含代码内容（不要留空白页）

## 注意事项

1. **源代码格式**
   - 所有代码都已格式化，包含行号
   - 保持原有缩进和格式
   - 不要修改代码内容

2. **文件完整性**
   - 确保前后30页都完整
   - 检查页码是否连续
   - 确保没有缺失页面

3. **打印质量**
   - 确保打印清晰
   - 避免墨迹模糊
   - 检查是否有缺页

## 重新生成文档

如果需要重新生成源代码文档，运行：

```bash
node scripts/generate-source-doc.js
```

这将重新生成所有源代码文档。
