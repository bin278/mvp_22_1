## 三、软件功能概述

mornFront 是一款基于人工智能技术的智能前端代码生成系统。该系统能够根据用户输入的自然语言描述，自动生成完整的、生产就绪的React/TypeScript前端应用程序代码。

### 3.1 核心功能

#### 3.1.1 智能代码生成
- **自然语言输入**：用户通过自然语言描述所需的前端界面和功能
- **模板智能识别**：系统自动识别用户需求，选择合适的项目模板（仪表板、落地页、表单、定价页、博客、游戏等）
- **完整项目生成**：自动生成包含所有必要文件的完整项目结构，包括：
  - package.json（依赖管理）
  - TypeScript配置文件
  - Vite构建配置
  - Tailwind CSS配置
  - React组件代码
  - 样式文件
  - README文档
  - .gitignore文件

#### 3.1.2 实时预览功能
- **即时预览**：生成的代码可以立即在浏览器中预览
- **交互式预览**：支持完整的交互功能测试
- **响应式预览**：预览支持多种屏幕尺寸

#### 3.1.3 项目下载功能
- **ZIP打包下载**：一键下载完整的项目文件
- **项目结构完整**：包含所有必要的配置文件和依赖信息
- **即用性**：下载后可直接运行，无需额外配置

#### 3.1.4 用户认证系统
- **多种登录方式**：支持邮箱密码注册登录、微信登录等第三方登录
- **会话管理**：安全的用户会话管理和JWT令牌处理
- **密码重置**：支持通过邮件重置密码

#### 3.1.5 GitHub集成
- **代码推送**：生成的代码可以直接推送到GitHub仓库
- **OAuth认证**：安全的GitHub OAuth认证流程
- **仓库管理**：创建和管理GitHub仓库

#### 3.1.6 支付订阅系统
- **订阅管理**：支持多层级订阅方案（免费、基础、专业、企业）
- **支付集成**：集成微信支付和支付宝支付
- **使用量统计**：跟踪和统计用户的使用情况

#### 3.1.7 对话历史管理
- **历史记录**：保存用户的生成历史记录
- **项目管理**：管理和查看之前生成的项目
- **文件管理**：查看和管理生成的项目文件

#### 3.1.8 多语言支持
- **双语界面**：支持中文和英文界面切换
- **国际化**：完整的i18n国际化支持

### 3.2 支持的模板类型

1. **AI工具网站模板**：适用于AI工具类网站，包含交互式生成、现代化设计等
2. **仪表板模板**：适用于数据分析和业务监控场景

## 四、软件使用说明

### 4.1 软件安装

#### 4.1.1 环境准备
1. 安装 Node.js（v18或更高版本）
2. 安装包管理器（pnpm或npm）

#### 4.1.2 安装步骤
```bash
# 克隆项目仓库
git clone <repository-url>
cd mvp_22

# 安装依赖
pnpm install
# 或
npm install
```

#### 4.1.3 配置环境变量
创建 `.env.local` 文件，配置必要的环境变量：
- DeepSeek API密钥
- 腾讯云CloudBase配置
- 支付相关配置（如需要）

### 4.2 启动运行

#### 4.2.1 开发模式
```bash
# 启动开发服务器
pnpm dev
# 或
npm run dev
```

访问地址：http://localhost:3000

#### 4.2.2 生产模式
```bash
# 构建生产版本
pnpm build

# 启动生产服务器
pnpm start
```

### 4.3 基本操作流程

#### 4.3.1 生成前端代码
1. 访问代码生成页面（/generate）
2. 在输入框中输入所需的前端界面描述（支持中英文）
3. 点击"生成代码"按钮
4. 等待系统生成代码（通常需要几秒钟）
5. 在文件浏览器中查看生成的代码文件
6. 点击"预览"按钮查看实时效果
7. 点击"下载"按钮下载完整的项目文件

#### 4.3.2 用户注册登录
1. 点击"注册"按钮
2. 填写邮箱和密码
3. 完成邮箱验证（如启用）
4. 登录系统开始使用

#### 4.3.3 管理生成的项目
1. 在用户中心查看历史生成记录
2. 选择项目查看详细信息
3. 重新下载或删除项目

#### 4.3.4 GitHub集成使用
1. 在设置页面绑定GitHub账户
2. 生成代码后选择"推送到GitHub"
3. 输入仓库名称和描述
4. 系统自动创建仓库并推送代码

### 4.4 高级功能

#### 4.4.1 订阅管理
- 查看当前订阅状态
- 升级或降级订阅方案
- 查看使用量统计

#### 4.4.2 支付功能
- 选择订阅方案
- 通过微信支付或支付宝完成支付
- 查看支付记录

## 五、软件技术特点

### 5.1 架构设计
- **前后端分离**：采用Next.js全栈框架，支持服务端渲染和客户端渲染
- **API优先**：RESTful API设计，便于扩展和集成
- **模块化设计**：代码模块化，便于维护和扩展

### 5.2 代码质量
- **TypeScript**：完整的类型系统，提供类型安全保障
- **代码规范**：遵循现代前端开发最佳实践
- **可维护性**：清晰的代码结构和注释

### 5.3 性能优化
- **服务端渲染（SSR）**：提升首屏加载速度
- **代码分割**：按需加载，减少初始包大小
- **CDN加速**：静态资源CDN分发
- **缓存策略**：合理的缓存机制

### 5.4 安全性
- **JWT认证**：安全的用户认证机制
- **HTTPS支持**：数据传输加密
- **输入验证**：完善的输入验证和过滤
- **权限控制**：细粒度的权限管理

### 5.5 可扩展性
- **模板系统**：易于扩展新的项目模板
- **插件机制**：支持功能插件扩展
- **API扩展**：标准的API接口设计
